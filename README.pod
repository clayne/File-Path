=head1 NAME

File::Path - Create or remove directory trees

=head1 INSTALLATION

    perl Makefile.PL
    make
    make test
    make install

=head1 PREREQUISITES

All prerequisites are found in the Perl 5 core distribution.

=head2 To configure:

    ExtUtils::MakeMaker

=head2 To build:

    Cwd
    Exporter
    File::Basename
    File::Spec

=head3 To test:

    Carp
    Config
    Cwd
    Errno
    Fcntl
    File::Spec::Functions
    SelectSaver
    Cwd
    Test::More
    Win32                   # Windows only
